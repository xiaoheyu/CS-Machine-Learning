The dataset used for part two is from UCI Machine Learning Repository
The name of the dataset is "Blood Transusion Service Center Dataset". 
This dataset is used for classification purpose. 
Totally 748 instances are included. 4 features and 1 target variable. 
Features is explained in the transfusion.names.txt file.  
The target is to predict whether a persion will donote the blood based on his/her histaoral donote records
